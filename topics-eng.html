Summer Club

Keep cool with us this holiday at our Summer Club! From June 17 to July 26, we will be opening our doors for another batch of Rookies! Our Mentors will be building and teaching curriculums for a variety of topics that you can choose from. Although the Club runs for six weeks, you can pick and choose which weeks you want to join and which topics to learn each week. Pssst, this Club is free you know ;)

Build a Website

Ever wondered what it takes to build a website? Join us to learn the basics of HTML, CSS, and JavaScript. You’ll learn step-by-step how to build an interactive website from a simple “Hello World!” to integrating third-party tools like Google Maps. By the end of the week, you will have built an entire website from scratch!


Great to pair with: Backend for Beginners, Build a mobile app, Web and App design.

Backend for Beginners

Building a web interface is only half the fun! In this topic, you’ll learn how to handle user input, store information in databases, and communicate back to the users accordingly. You’ll get a behind-the-scenes look at the backend logic of a website!


Great to pair with: Build A Website, Build a Mobile App

Build a Mobile App

Have an idea for the next big app that will have everyone glued to their phones? Join us to learn the basics of Android mobile development in Java! From how to layout buttons and text to navigating between different screens, this topic will get you started on publishing your own app to the app store ;)


Great to pair with: Build A Website, Backend for Beginners, Web and App Design

Computer Science 101

Before you can walk, you have to learn to crawl. We will be using Python to learn basic concepts like for-loops (control flow) and object-oriented programming to give you the foundation for more advanced concepts. CS 101 is the fundamentals topic that will teach you the technical concepts all successful Computer Scientists should know.


Great to pair with: Robotics, Build a Website

Web and App Design

This one is for all the designers out there! A user’s experience on a site or app is just as important as the features on it. This is a non-technical topic (very little coding involved) where you will learn how to create beautiful web and mobile app designs, interactive flows, and prototypes using Figma and Adobe Illustrator.


Great to pair with: Build a Website, Build a Mobile App

Robotics

Circuits, lights, and sensors, oh my! In this topic, we will be using Arduino boards and software to explore the wonderful world of robotics. From basic light installations and motion sensors to robotic pets that you can play games with, there are infinite possibilities on the different projects you can build!


Great to pair with: Computer Science 101
